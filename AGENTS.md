# AGENTS.md - RocketWatchDog.ai

## Overview

RocketWatchDog.ai is a policy/guard proxy for LLM + MCP traffic.

## Build / Run

```bash
npm install
npm run dev
```

## Tests

```bash
npm test
```

## Lint

```bash
npm run lint
```

## Conventions

- TypeScript strict mode
- Validate all external inputs
- Prefer small, explicit modules
- Keep config file-based (YAML + JSON schema)
